003 : EMI Calculation

This is web-based project. In this project we have created a web-based application using Python and Django framework.

  • Created users directly in the database.
  • In front end, user will have to enter its loan amount, disbursal amount (E.g.: 1,00,000 and 90,000 respectively.) and click on "Generate EMI" button. This should show the EMI amount to be paid as per Interest amount 18% of the loan amount. User will have to enter when the EMI amount is paid. If paid later than 5th of every month, an overdue of 18% to be paid additional to the EMI amount.
  • To calculate emi, overdue and total (sum of emi and overdue), we have used in-built @property decorator which will allow us to use model variables for performing calculations.
  • Generate a report showing summary of all 12 months showing EMI amount, date and overdue amount if applicable.
  • Used Django in-built Class Based Views to perform CRUD operations such as create a new user, update emi for existing user and delete user already created.
  • Used Django in-built Class Based Views to list all users and get details of each user.
